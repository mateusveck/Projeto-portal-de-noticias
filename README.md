# Portal de Notícias

Projeto de uma página de notícias sobre tecnologia, desenvolvido durante meus estudos de HTML e CSS.

## Sobre o projeto

A proposta foi criar um portal com notícias, categorias, destaques e uma estrutura parecida com sites reais de conteúdo.

Foi um projeto bom para praticar layout com várias seções, cards, imagens e organização visual da página.

## Tecnologias usadas

- HTML
- CSS
- Git
- GitHub

## O que pratiquei

- Layout de portal de notícias
- Organização de conteúdo em cards
- Uso de imagens
- Grid e espaçamentos
- Estruturação de página com várias seções

## Status

Finalizado, com possibilidade de melhorias.
